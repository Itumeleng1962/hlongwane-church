# Changing Minds Prophetic Ministries (Hlongwane Church)

Welcome to the official web platform for Changing Minds Prophetic Ministries (National Army). This application is a comprehensive digital footprint designed to support the church's outreach, job creation, global missions, and spiritual leadership under Apostle Msizi Hlongwane.

## About the Application
This is a modern web application designed with an elegant dark theme and gold accents to reflect the premium, structured, and divine nature of the church. 

Features include:
- Home & About Pages: Highlighting the mission, vision, and core values of the ministry.
- Leadership Directory: Outlining the board and visionary leadership driving the church.
- Sermons & Blog: Digital access to recent messages and devotionals.
- Ministries & Impact: Information on outreach, job creation programs (SETA/NQF aligned), and community empowerment.
- Giving/Donations: Streamlined information for partnering with the ministry.
- Interactive Routing: Fast and robust page navigation using Next.js.

## Tech Stack
- Frontend Framework: Next.js (App Router)
- Language: TypeScript 
- Styling: Tailwind CSS & Custom globals.css
- Icons: Lucide-React
- Animations: Framer Motion

## Getting Started
